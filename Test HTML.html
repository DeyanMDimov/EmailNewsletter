MEETUP 2/24 
CHICAGO .NET MOBILE DEVELOPERS
FreshMvvm : A Lightweight MVVM Framework for Xamarin.Forms

Tue. 2/24 5:30PM
222 W Adams St
There are several MVVM frameworks available that are compatible with Xamarin PCLs, but most either have a heavyweight implementation, or are cumbersome to use with Xamarin.Forms. FreshMvvm is a new MVVM Framework whose single purpose is to make it easy to write code in a MVVM style for a Xamarin.Forms app. Calvin Fisher from InContext Solutions will introduce the basic concepts of the FreshMvvm library, and will review a sample project to demonstrate how it can be used in in a Xamarin.Forms application.
